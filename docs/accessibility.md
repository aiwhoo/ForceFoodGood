## Accessibility

This project follows basic accessibility guidelines to ensure usability for all users:

- All interactive elements include descriptive labels.
- Color choices meet WCAG contrast requirements.
- Keyboard navigation is supported for all major features.
- Images include alt text where appropriate.
- Error messages are readable and not color‑dependent.
- Layout is responsive and readable on various screen sizes.

Additional improvements could include ARIA roles and screen‑reader testing.
